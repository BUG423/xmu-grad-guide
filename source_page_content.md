## 特别说明

1. 错过每批次要求的论文答辩截止时间、纸质材料提交截止时间的，均只能申请下一批次的授学位。
2. 请认真仔细阅读全文，严格按要求进行。所有不按要求规范提交的材料或申请，均退回重新提交。
3. 学位申请各流程具体要求，请详见《信息学院（国家示范性软件学院）博士学位和硕士学位授予工作实施细则（试行）》[信院（2024）14号]。
4. 本研系统（https://jw.xmu.edu.cn/new/index.html）毕业、申请学位操作指南：
   - 学生版操作指南：https://docs.qq.com/doc/DRWhid0tTV1VSUlpL
   - 英文版：https://docs.qq.com/doc/DRUxnWEhuUHV0aGRi
5. 所有流程中，需要"导师同意"或"导师签字同意"的，普通学生须经主导师同意；联培学生须经双方导师同意。
6. 以实践成果申请学位的专业型研究生，除特殊说明外，流程参考以学位论文申请学位的流程。

## 毕业论文送审、答辩及申请学位流程简表

### 步骤一：定稿送审

| 具体流程节点 | 注意事项 |
|------------|---------|
| 完成毕业生毕业图像采集 | 注意通知并及时完成。该图像用于毕业证和学位证，以及学信网 |
| 1. 科研成果审核 | 之前已审核过则无须重复进行 |
| 2. 论文定稿 | 定稿须经导师同意；联培学生须经双方导师同意 |
| 3. 预答辩并修改再次定稿 | **博士生必须完成** |
| 4. 系统上进行送审申请 | 仅学分全部完成、预答辩完成（博士生）方有权限进行申请；毕业图像采集未完成的同学也无权限申请 |
| 5. 定稿论文查重（第一次查重） | 邮件须附上《答辩申请表》 |
| 6. 定稿论文送审前形式审查 | 审查电子版 |
| 7. 定稿论文送审 | 全部走教育部平台送审（国际学生除外） |
| 8. 实践成果报告鉴定会 | 以实践成果申请学位的专业型研究生必须完成 |

### 步骤二：答辩

| 具体流程节点 | 注意事项 |
|------------|---------|
| 1. 博士论文质量核查 | **博士生必须完成** |
| 2. 修改论文，做好答辩准备 | |
| 3. 到系统维护完善个人相关信息 | 《研究生登记表》《毕业生登记表》等档案材料均由系统打印生成 |
| 4. 依据要求进行答辩 | 请务必在每批次要求的答辩截止时间前完成答辩 |

### 步骤三：答辩后提交材料

| 具体流程节点 | 注意事项 |
|------------|---------|
| 1. 修改论文，再次定稿 | 定稿须经导师同意；联培学生须经双方导师同意 |
| 2. 定稿论文查重（第二次查重） | 邮件须附上《答辩决议》扫描件 |
| 3. 系统里进行学位申请 | 打印出《学位申请表》 |
| 4. 打印毕业论文，提交所有纸质材料进行论文形式审查 | 注意每批次截止时间；纸质版论文通过形式审查后按时交全所需份数 |

## 详细流程

### 一、毕业论文送审流程

#### （一）科研成果（创新性成果）审核
（系统上已完成审核的，可以跳过此步骤）

1. 准备材料：按《博士生、硕士生申请学位所需的科研成果检索要求及系统填写注意事项》的要求，提供科研成果的证明材料（图书馆检索证明+论文全文等）
   - 注意事项可见附件2压缩包，或网页：https://informatics.xmu.edu.cn/info/2301/178001.htm
2. 系统上填写科研成果，并请导师在系统上审核通过
3. 通过邮件提醒相关研究生秘书进行审核，邮件名：科研成果审核（专利/论文/竞赛）-姓名-学号
4. 注意事项：该导师签字的，务必导师手写签名后扫描或拍照

#### （二）学分及前置学历审核

1. 在研究生系统上检查学分是否全部完成
2. 检查前置学历学位是否填写完整且通过审核
3. 未通过审核的，及时联系相关研究生秘书

#### （三）论文定稿（含预答辩要求）

**1. 博士预答辩**
- 博士生首次定稿后必须进行预答辩，一般应安排在正式答辩的3个月前进行
- 预答辩通过后方可安排论文送审
- 硕士生是否要组织预答辩，由导师（导师组）或实验室确定
- 申请预答辩前应满足：
  ① 修完全部课程，完成规定的培养环节并取得规定的学分
  ② 学位论文已完稿并经导师签字同意
  ③ 导师签字同意其参加预答辩
  ④ 科研成果（创新性成果）达到要求
- 集中预答辩流程：
  ① 导师同意定稿，同意进行预答辩
  ② 博士生在本研系统中提交预答辩申请
  ③ 学院对预答辩信息进行审核，并公布预答辩安排表
  ④ 由学院统一组织预答辩
- 预答辩表格：①厦门大学研究生学位论文预答辩申请表 ②预答辩成绩评定表

**2. 论文字数要求**
- 硕士学位论文：一般不少于3万字
- 博士学位论文：一般不少于5万字（英文博士论文一般要求80页左右——仅针对国际博士生）
- 特别注意：论文格式严格按照厦门大学毕业论文规范要求（附件5）

#### （四）本研系统进行送审申请

1. 进入本研系统进行送审申请，选择线上审核，请导师到系统审核
2. 导出《送审申请表》PDF电子版，打印有导师签字的纸质版
3. 系统里填写论文相关信息时，严格按照字数要求，不要超过限定字数

#### （五）论文查重

1. 查重论文命名格式：学号_作者_篇名_专业_导师
2. 论文查重有两次：第一次送审前，第二次答辩后
3. **第一次查重（送审论文查重）**：
   - 格式为PDF，全文报送查重（不包括原创性声明和著作权使用声明）
   - 必须小于20M
   - 发送至相关负责老师邮箱，抄送导师，附上《送审申请表》
   - 邮件标题：论文送审前查重-学号-姓名-专业-联系电话
   - 博士生注意：仅当通过预答辩，方可进行定稿论文的查重申请
4. **第二次查重（答辩后终稿论文查重）**：
   - 终稿论文全文报送查重（包含论文全部组成部分）
   - 删除文中空白页，小于20M
   - 发送至相关负责老师邮箱，抄送导师
   - 邮件标题：答辩后定稿查重-学号-姓名-专业-联系电话
   - 附上《学位论文答辩决议》扫描件
   - 只有定稿查重通过后，方可在系统上进行学位申请
5. 查重率低于10%方可进行学位申请
6. 未按规定格式保存和命名拟查重的论文，将不予以进行查重

#### （六）论文送审

1. 所有的论文均须进行盲审
2. 盲审论文必须删除：学生姓名、导师姓名、致谢、发表的科研论文的作者部分等
3. 盲审论文提交给相关研究生秘书进行送审
4. 研究生秘书在送审前进行论文形式审查，未通过将予以退回修改
5. 评阅人数要求：
   - 硕士学位论文评阅人应不少于2名
   - 博士学位论文评阅人应不少于3名
   - 专业型研究生的学位论文评阅，有一篇需要由行（企）业专家评审

**教育部平台送审（全体国内硕博研究生）**：
- 提交材料：盲审论文电子版(PDF) + 论文信息汇总表(excel) + 摘要(TXT)
- 盲审论文命名：10384_学号_LW (如：10384_100012013120001_LW.pdf，学号中字母务必大写)
- 论文信息汇总表命名：10384_学号-姓名-专业-论文信息汇总表
- 摘要命名：10384_学号_ZY（如：10384_10011201312002_ZY.txt，中文即可）
- 平台评审周期：博士35个工作日，硕士25个工作日；增评顺延15个工作日
- 送审时间节点：6月授学位应于3月底4月初提交，12月授学位应于9月底10月初提交

**学院送审（国际研究生、专业学位研究生）**：
- 校外送审时长：20天
- 专业学位硕士：登录学院系统（https://jwgs.xmu.edu.cn/H5/）提交盲审论文
- 国际研究生：盲审论文电子版(PDF+全文加水印)，水印内容为学号

**评阅专家要求**：
- 博士：需提供6人备选名单，均为校外专家（建议均为省外），博导或正高级
- 硕士：需提供5人备选名单，均为校外高级职称硕导，省外盲审须全部为外省
- 专业学位：须有相关行（企）业专家不少于三分之一

**论文评阅书结果要求**：
- 评阅分数≥75分：可直接进入论文答辩环节
- 70分≤评阅分数<75分：修改30天后，提交论文修改说明，经审核批准后方可答辩
- 评阅分数<70分：
  - 1名：增聘2名评阅人；修改不少于30天；增评均≥70分可答辩
  - 2名或以上：不可进入论文答辩环节
- 首次送审至少1篇不合格：至少6个月后才能重新送审
- 第二次送审至少1篇不合格：至少1年后才能重新送审
- 第三次送审仍不合格：不再受理学位论文送审申请

**博士论文质量核查**：
- 博士必须完成，采用线下专家双向匿名评审
- 核查内容：论文创新性、应用价值、文献综述、结果分析、写作规范
- 核查比例不低于评审论文数的20%
- 要求修改的论文，修改完成后提交《博士学位论文质量核查结果反馈》

**论文修改说明**：
- 所有评阅书收回后，在导师指导下依据评审意见修改论文
- 纸质版论文修改说明需本人手写签名，导师同意后签名确认
- 应在答辩时提交给答辩委员会参考

### 二、答辩准备

1. 再次确认科研成果审核完成
2. 确定答辩委员会成员、答辩秘书；导师将答辩秘书名单发送给研究生秘书
3. 准备若干份打印好的答辩论文（答辩委员会成员数+导师1本+秘书1本或本人1本）
   - 注意：此版非最终定稿，请勿打印过多；差错率不得高于万分之二
4. 答辩秘书提前至少5天登录本研系统填写答辩安排信息
5. 研究生秘书打印答辩审批表，在系统报送答辩审批
6. 答辩秘书领取答辩档案袋，打印《学位论文评定书》《答辩情况记录表》《答辩决议》
   - 注：《学位论文优秀推荐决议书》仅论文推荐为"优秀"的同学填写
7. 博士生须制作答辩海报PPT，在答辩前2个工作日发送至研究生秘书邮箱

---

### 三、答辩

1. 答辩时答辩秘书进行记录和录音（硕、博均需要）
2. 答辩通过后，依据答辩委员会意见修改论文，导师审核同意定稿后再次提交查重
3. 学位申请：在研究生系统里进行学位申请，打印出《学位申请表》2份，上传论文终稿
